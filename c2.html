Chapter 2: Fundamentals of Operating System

Complete Notes, MCQs, Questions & Important Points

Chapter Notes

What is an Operating System?

An Operating System (OS) is system software that manages computer hardware, software resources, and provides services to users and programs.

Functions of an Operating System

  • 1. Memory Management: Manages RAM and allocates memory to programs.
  • 2. Process Management: Controls running programs and multitasking.
  • 3. File Management: Controls files, folders, storage devices.
  • 4. Device Management: Manages input/output devices.
  • 5. Security: Protects data with passwords and permissions.
  • 6. User Interface: Provides GUI or command interface.

Types of Operating Systems

  • Single User OS: Used by one person at a time (e.g., Windows 10).
  • Multi-User OS: Many users access the system (e.g., Linux server).
  • Real-Time OS: Used in critical systems (e.g., Airplane control).
  • Batch Processing OS: Processes jobs in batches.
  • Distributed OS: Controls multiple connected computers.

Examples of Operating Systems

  • Microsoft Windows
  • Linux
  • Android
  • MacOS
  • Unix

MCQs

1. Operating System is a:

a) Hardware
b) System Software
c) Application Software
d) None

Answer: b) System Software

2. Which OS is open-source?

a) Windows
b) Linux
c) MacOS
d) Android

Answer: b) Linux

3. GUI stands for:

a) General User Interface
b) Graphical User Interface
c) Graph Unit Interface
d) Global User Interaction

Answer: b) Graphical User Interface

4. The core part of OS is called:

a) Shell
b) Kernel
c) Interface
d) Hardware

Answer: b) Kernel

5. OS manages:

a) Files
b) Memory
c) Devices
d) All of these

Answer: d) All of these

Short Questions

  1. What is an Operating System?
    Answer: System software that manages hardware and software resources.
  2. What is a Kernel?
    Answer: The core part of the OS that controls everything.
  3. What is GUI?
    Answer: Graphical User Interface.
  4. Define multitasking.
    Answer: Ability of OS to run multiple programs at the same time.
  5. Give two examples of OS.
    Answer: Windows, Linux.

Long Questions

  1. Explain the functions of an Operating System.
    Answer: OS performs memory management, process management, file management, device control, security, and provides user interface.
  2. Describe the types of Operating Systems.
    Answer: Single-user, multi-user, batch, real-time, and distributed OS.
  3. Write a note on GUI and CLI.
    Answer: GUI uses icons and windows; CLI uses text commands.
  4. Explain why OS is important for a computer.
    Answer: Without OS, a computer cannot run programs, manage hardware, or interact with users.

Important Points

  • OS is system software.
  • Two main interfaces: GUI and CLI.
  • Main functions: memory, files, devices, processes, security.
  • Examples: Windows, Linux, Android, MacOS.
