Chapter 2: Logarithms

Comprehensive Notes, Questions, and Practice for Sindh Textbook Board

Chapter Notes

This chapter covers the concept of logarithms, their properties, and applications in simplifying calculations.

Introduction to Logarithms

Definition

If ax = N, then logarithm of N to the base a is x, written as logaN = x

Where: a > 0, a ≠ 1, and N > 0

Examples

23 = 8 ⇒ log28 = 3

102 = 100 ⇒ log10100 = 2

5-2 = 1/25 ⇒ log5(1/25) = -2

Fundamental Laws of Logarithms

Product Law

loga(m × n) = logam + logan

Quotient Law

loga(m/n) = logam - logan

Power Law

logamn = n × logam

Base Change Formula

logab = logcb / logca

Special Cases and Important Formulas

  • loga1 = 0 (since a0 = 1)
  • logaa = 1 (since a1 = a)
  • logaax = x
  • alogax = x
  • logab × logba = 1
  • logab = 1 / logba

Common Logarithms and Natural Logarithms

  • Common Logarithm: Base 10, written as log10x or simply log x
  • Natural Logarithm: Base e (e ≈ 2.718), written as logex or ln x

Characteristic and Mantissa

For common logarithms (base 10):

  • Characteristic: Integral part of the logarithm
  • Mantissa: Decimal part of the logarithm (always positive)
Example

log10250 = 2.3979

Characteristic = 2, Mantissa = 0.3979

Using Logarithm Tables

Steps to find logarithm using tables:

  1. Determine the characteristic
  2. Find mantissa from the log table
  3. Combine characteristic and mantissa
Number Characteristic Reason
Numbers ≥ 1 Positive Number of digits before decimal - 1
Numbers between 0 and 1 Negative Number of zeros immediately after decimal

Multiple Choice Questions (MCQs)

1. What is log101000 equal to?

a) 2
b) 3
c) 4
d) 10

Answer: b) 3 (since 10³ = 1000)

2. If log2x = 5, what is the value of x?

a) 10
b) 25
c) 32
d) 64

Answer: c) 32 (since 2⁵ = 32)

3. What is loga1 equal to?

a) 0
b) 1
c) a
d) undefined

Answer: a) 0 (since a⁰ = 1)

4. Which law is represented by: log(mn) = log m + log n?

a) Power law
b) Quotient law
c) Product law
d) Base change law

Answer: c) Product law

5. What is the characteristic of log 0.005?

a) 0
b) -2
c) -3
d) 3

Answer: c) -3 (two zeros after decimal: 0.005 = 5 × 10⁻³)

6. If log 2 = 0.3010, what is log 20?

a) 1.3010
b) 2.3010
c) 0.6020
d) 1.6990

Answer: a) 1.3010 (log 20 = log(2×10) = log 2 + log 10 = 0.3010 + 1)

7. What is the value of log327?

a) 2
b) 3
c) 4
d) 9

Answer: b) 3 (since 3³ = 27)

8. Which is equivalent to logab × logba?

a) 0
b) 1
c) a
d) b

Answer: b) 1

9. What is the base of natural logarithms?

a) 2
b) 10
c) e
d) π

Answer: c) e (Euler's number ≈ 2.718)

10. If log x = 2.7531, what is the characteristic?

a) 0.7531
b) 2

Answer: b) 2 (the integer part before decimal)

Short Questions

  1. Define logarithm with example.
    Answer: If aˣ = N, then logₐN = x. Example: 2³ = 8 ⇒ log₂8 = 3.
  2. State the product law of logarithms.
    Answer: logₐ(m × n) = logₐm + logₐn
  3. What is the value of logₐa?
    Answer: 1 (since a¹ = a)
  4. Differentiate between characteristic and mantissa.
    Answer: Characteristic is the integer part, mantissa is the decimal part of logarithm.
  5. Find the characteristic of log 0.0007.
    Answer: -4 (three zeros after decimal: 0.0007 = 7 × 10⁻⁴)
  6. State the power law of logarithms.
    Answer: logₐmⁿ = n × logₐm
  7. What is the value of log 1?
    Answer: 0 (since 10⁰ = 1)
  8. Convert 4³ = 64 to logarithmic form.
    Answer: log₄64 = 3
  9. What is common logarithm?
    Answer: Logarithm with base 10, written as log x.
  10. Simplify: log 5 + log 2
    Answer: log(5 × 2) = log 10 = 1

Long Questions

  1. Explain the fundamental laws of logarithms with examples.
    Answer: Product law: log(mn)=log m+log n (log(10×100)=log10+log100=1+2=3). Quotient law: log(m/n)=log m-log n (log(100/10)=log100-log10=2-1=1). Power law: log mⁿ=n log m (log100²=2log100=2×2=4).
  2. Describe characteristic and mantissa with examples.
    Answer: Characteristic is the integer part representing the exponent, mantissa is the decimal part from log tables. Example: log 250 = 2.3979, characteristic=2, mantissa=0.3979. For numbers less than 1, characteristic is negative but mantissa is always positive.
  3. Prove that logₐb × logₐa = 1.
    Answer: Let logₐb = x ⇒ aˣ = b, and logₐa = 1. Then logₐb × logₐa = x × 1 = x. But x = logₐb, and logₐb = 1/logₐa, so logₐb × logₐa = (1/logₐa) × logₐa = 1.
  4. Explain how to use logarithm tables to find logarithms of numbers.
    Answer: Step 1: Write number in scientific notation. Step 2: Determine characteristic from exponent. Step 3: Find mantissa from log table using significant digits. Step 4: Combine characteristic and mantissa. Step 5: Apply laws for intermediate calculations.
  5. Discuss the applications of logarithms in real life.
    Answer: Logarithms are used in: 1) Scientific calculations (simplifying multiplication/division), 2) Measuring earthquake intensity (Richter scale), 3) Sound intensity (decibels), 4) pH calculations in chemistry, 5) Computer science algorithms, 6) Financial calculations (compound interest).

1. Express in logarithmic form: 5³ = 125

Solution: log₅125 = 3

2. Simplify: log 8 + log 125

Solution: log 8 + log 125 = log(8 × 125) = log 1000 = 3

3. Find the value of: log₂16

Solution: 2⁴ = 16, so log₂16 = 4

4. Simplify: log 75 - log 3

Solution: log 75 - log 3 = log(75/3) = log 25 = log 5² = 2 log 5

5. Find the characteristic of: (a) log 4750 (b) log 0.038

Solution: (a) 4750 = 4.75 × 10³, characteristic = 3
(b) 0.038 = 3.8 × 10⁻², characteristic = -2

6. Prove that: logₐ(m/n) = logₐm - logₐn

Solution: Let logₐm = x ⇒ aˣ = m, and logₐn = y ⇒ aʸ = n. Then m/n = aˣ/aʸ = aˣ⁻ʸ. So logₐ(m/n) = x - y = logₐm - logₐn

7. If log 2 = 0.3010 and log 3 = 0.4771, find log 6

Solution: log 6 = log(2 × 3) = log 2 + log 3 = 0.3010 + 0.4771 = 0.7781

8. Simplify: 2 log 5 + 3 log 2

Solution: 2 log 5 + 3 log 2 = log 5² + log 2³ = log 25 + log 8 = log(25 × 8) = log 200

Important Points

  • Understand the definition: aˣ = N ⇔ logₐN = x
  • Memorize all fundamental laws: Product, Quotient, and Power laws
  • Learn special cases: logₐ1 = 0, logₐa = 1, logₐaˣ = x
  • Understand characteristic and mantissa for common logarithms
  • Practice converting between exponential and logarithmic forms
  • Learn to use logarithm tables for calculations
  • Remember base change formula: logₐb = logₓb / logₓa
  • Know the difference between common log (base 10) and natural log (base e)
