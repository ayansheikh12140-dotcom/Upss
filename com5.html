Chapter 5: Computer Security and Ethics

Comprehensive Notes, Questions, and Practice for Sindh Textbook Board

Chapter Notes

This chapter covers computer security threats, protection measures, and ethical issues in computing.

What is Computer Security?

Computer security refers to protection of computer systems and data from harm, theft, and unauthorized use.

Types of Security Threats

Malware (Malicious Software)
  • Virus: Self-replicating program that attaches to other programs
  • Worm: Standalone malware that replicates itself to spread
  • Trojan Horse: Disguised as legitimate software but performs malicious actions
  • Spyware: Secretly monitors user activities
  • Ransomware: Locks system and demands payment
Other Threats
  • Hacking: Unauthorized access to computer systems
  • Phishing: Fraudulent attempts to obtain sensitive information
  • Data Theft: Stealing confidential information
  • Denial of Service (DoS): Overwhelming system to make it unavailable

Security Measures

  • Antivirus Software: Detects and removes malware
  • Firewall: Blocks unauthorized network access
  • Encryption: Converts data into unreadable code
  • Strong Passwords: Complex and regularly changed
  • Backup: Regular copies of important data
  • Access Control: Restricting user permissions

Computer Ethics

Ethical guidelines for computer usage and professional conduct.

Key Ethical Principles
  • Respect for privacy and confidentiality
  • Honesty and integrity in computer use
  • Respect for intellectual property rights
  • Social responsibility in technology use
  • Avoiding harm to others through computing

Intellectual Property Rights

  • Copyright: Legal protection for original works
  • Patent: Protection for inventions
  • Trademark: Protection for brands and logos
  • Software Piracy: Illegal copying/distribution of software

Cyber Laws in Pakistan

  • Electronic Transactions Ordinance (2002): Legal recognition of electronic documents
  • Prevention of Electronic Crimes Act (2016): Addresses cybercrimes
  • Data Protection Laws: Protection of personal data

Safe Computing Practices

  • Use updated antivirus software
  • Avoid suspicious emails and links
  • Use strong, unique passwords
  • Regular software updates
  • Be cautious on social media
  • Regular data backups

Short Questions

  1. Define computer security.
    Answer: Protection of computer systems and data from harm, theft, and unauthorized access.
  2. What is a computer virus?
    Answer: A malicious program that attaches itself to other programs and replicates.
  3. Differentiate between virus and worm.
    Answer: Virus needs a host program, while worm is standalone and self-replicating.
  4. What is phishing?
    Answer: Fraudulent attempt to obtain sensitive information by pretending to be trustworthy.
  5. Name three security measures.
    Answer: Antivirus software, firewall, encryption, strong passwords (any three).
  6. What is computer ethics?
    Answer: Moral principles governing the use of computers and digital technology.
  7. Define software piracy.
    Answer: Illegal copying, distribution, or use of software without proper license.
  8. What is the purpose of copyright?
    Answer: To protect original creative works from unauthorized use.
  9. Name two cyber laws in Pakistan.
    Answer: Prevention of Electronic Crimes Act, Electronic Transactions Ordinance.
  10. What is a strong password?
    Answer: A password with mix of letters, numbers, symbols, and sufficient length.

Long Questions

  1. Explain different types of malware with examples.
    Answer: Virus (attaches to programs), Worm (self-replicating standalone), Trojan Horse (disguised as legitimate), Spyware (monitors secretly), Ransomware (locks system for payment). Examples: ILOVEYOU virus, Conficker worm, Zeus Trojan.
  2. Describe various computer security measures.
    Answer: Antivirus software detects and removes malware. Firewall blocks unauthorized network access. Encryption converts data to unreadable form. Strong passwords prevent unauthorized access. Regular backups protect against data loss. Access control restricts user permissions.
  3. Discuss the importance of computer ethics in modern society.
    Answer: Computer ethics ensure responsible technology use, protect privacy, prevent harm, respect intellectual property, and promote social responsibility. They guide behavior in areas like data handling, software development, and online interactions.
  4. Explain intellectual property rights related to computing.
    Answer: Copyright protects software, documents, and digital content. Patent protects computer inventions and algorithms. Trademark protects software brands and logos. These rights encourage innovation while protecting creators' interests.
  5. What are safe computing practices? Explain with examples.
    Answer: Using updated antivirus (protection against malware), avoiding suspicious links (prevents phishing), strong passwords (prevents unauthorized access), regular updates (fixes security vulnerabilities), cautious social media use (protects privacy), regular backups (prevents data loss).

Security Scenario Problems

1. A company has 100 employees. Each needs a unique password of at least 8 characters. Calculate the minimum time to crack one password if a hacker can try 1000 passwords per second (assuming lowercase letters only).

Solution: 26 lowercase letters, 8 characters = 26^8 = 208,827,064,576 combinations. Time = 208,827,064,576 / 1000 = 208,827,064 seconds ≈ 6.6 years.

2. If a virus infects 10 computers each hour, how many computers will be infected after 24 hours?

Solution: Starting with 1 infected computer, after 1 hour: 10 infected, after 24 hours: 10^24 = 1,000,000,000,000,000,000,000,000 computers (theoretical maximum).

3. A backup system creates copies every 4 hours. How much data could be lost maximum if system fails?

Solution: Maximum data loss = 4 hours of work. If backups are done at 12:00, 4:00, 8:00, etc., and system fails at 3:59, all work since last backup at 12:00 is lost.

4. Calculate the strength difference between 6-character and 8-character passwords with mixed characters.

Solution: Assuming 26 lowercase + 26 uppercase + 10 digits + 10 symbols = 72 characters. 6-char: 72^6 = 139,314,069,504. 8-char: 72^8 = 722,204,136,308,736. 8-char is about 5,184 times stronger.

5. A company loses $5000 per hour of downtime. How much would a 8-hour system failure cost?

Solution: Cost = 8 hours × $5000/hour = $40,000 total loss from system downtime.
